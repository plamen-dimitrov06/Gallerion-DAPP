Gallerion - DApp Gallery

    Welcome to the "Gallerion" DApp. It is a fully-functional Decentralized Application consisting of Solidity smart contract in the Ethereum blockchain + client-side app (HTML5) and server-side logic (JavaScript). The Gallerion keeps some data and logic in the Ethereum blockchain and provides UI to access it.

  • Each picture consists of JSON document plus the image in IPFS
  • Guests can view the photos and pictures
  • Guests can register as users and buy pictures
  • Users should have a MetaMask wallet
  • Authors register and list picures for sell in the Ethereum blockchain
